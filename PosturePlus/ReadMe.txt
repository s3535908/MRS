Purpose: This applications purpose is to present data from a SparkCore (Particle) in a graphical nice way. The application saves the nessesary information to access the SparkCore as CoreData. Based on this information its makes two JSON request to get the variable and then save it as CoreData. This is not working completely. The JSON request is only answered when the SparkCore is online - so you if you try to call, you will propably not receive any responce.

Inspiration to draw PieCharts: http://www.raywenderlich.com/90693/modern-core-graphics-with-swift-part-2

Works in Xcode 6.3.2.
